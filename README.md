# CustormerService---lading-Page
Este proyecto es una representación visual y conceptual de un sistema de gestión de soporte y atención al cliente. No se trata de una aplicación funcional; su objetivo es mostrar el diseño de interfaces UI/UX y la estructura de la plataforma como una landing page demostrativa.

La plataforma propuesta organiza la atención a clientes mediante tickets, seguimiento de casos, control de estados y un módulo de sugerencias y denuncias. Esta landing page sirve para:
<ul >
<li>Visualizar la estructura del dashboard con métricas del servicio.</li>
<li>Presentar los formularios de captura de datos (creación de tickets, envío de sugerencias o denuncias).</li>
<li>Mostrar los reportes y menús de la aplicación de manera conceptual.</li>
<li>Comunicar cómo sería la navegación y la interacción del usuario dentro del sistema.</li>
</ul>
# Características destacadas del diseño
<ul>
<li>Dashboard principal con métricas clave de soporte.</li>
<li>Panel de gestión de tickets y casos en diferentes estados (pendiente, en proceso, resuelto).</li>
<li>Módulo para recibir y organizar sugerencias y denuncias.</li>
<li>Formularios de entrada de datos claros y sencillos.</li>
<li>Ejemplos de menús de navegación y reportes generados.</li>
</ul>
